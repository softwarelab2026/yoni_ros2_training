to run the sim:
1. open new terminal > xhost +
2. then run the sim with: ros2 run [package-name] [node-name]

to run the turtlesim with control with keyboard:
ros2 run turtlesim turlesim_node
and then in new terminal:
ros2 run turtlesim turtle_teleop_key

to check the topic name for the cmd_vel you can run the sim and then open new terminal and run:
ros2 topic list
